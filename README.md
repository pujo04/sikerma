# Panduan Clone & Setup Project SIKERMA (Next.js)

Dokumen ini menjelaskan **langkah lengkap dari awal** untuk melakukan **clone repository**, **setup environment**, **install dependency**, hingga **menjalankan aplikasi dengan `npm run dev`** pada project **SIKERMA** yang dibangun menggunakan **Next.js**.

---

## 1. Prasyarat Sistem

Sebelum memulai, pastikan perangkat telah terpasang:

- **Node.js** (disarankan versi LTS ≥ 18)
- **Git**
- **npm** (atau pnpm sebagai alternatif)

### Cek Instalasi
Jalankan perintah berikut di terminal:

```bash
node -v
npm -v
git --version
